Execution:
1) Open the project folder in any IDE
2) Run the gui.java file: This file generates a Graphical USer Interface for the server and makes it listen to incoming client requests
3) Run client1.java, client2.java , client3.java individually in IDE
4) Server GUI window will show the users connected in realtime.
5) Clients have to pass messages through command line
